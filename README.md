# React 게시판 연습 프로젝트 🚀

## 📅 프로젝트 정보

- **시작일**: 2026-04-28
- **작성자**: PSV

## 📝 프로젝트 소개

6월에 예정된 Vue → React 전환 프로젝트를 대비한 연습용 저장소입니다.
React 기본 문법을 익히고 실무에서 바로 활용할 수 있는 역량을 기르는 것을 목표로 합니다.

- **기능**: 게시판 화면 구현 (목록, 상세, 작성 등)
- **특징**: 프론트엔드 화면만 구현 (백엔드 연동 X)

## 🛠 기술 스택

- **React** - UI 라이브러리
- **Create React App** - 개발 환경 및 빌드 도구
- **SCSS (Sass)** - 스타일링
- **react-quill** - 게시글 작성/수정 리치 텍스트 에디터

## 🌐 접속 주소

- **개발 서버**: http://localhost:3000 (포트 변경 시 아래 참고)

## ⚙️ 포트 변경

`react_project/.env` 파일에서 포트 번호 수정 후 `npm start` 재실행

```
PORT=4000
```

## ⚙️ 개발 환경

- **Node.js** v20.20.2
- **npm** 10.8.2

## 🚀 시작하기

```bash
# 패키지 설치
npm install

# 개발 서버 실행
npm start

# 빌드
npm run build
```
